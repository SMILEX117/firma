# Firmenserver-Simulation

Simulation einer typischen Verzeichnisstruktur eines Unternehmensservers.

## Struktur
* `users/`: Konfigurations- und Rechtedateien der Mitarbeiter.
* `scripts/`: Zentrale Automatisierungs- und Wartungsskripte.
* `backups/`: Ablageort für System- und Daten-Backups.
* `docs/`: Server- und Netzwerkdokumentationen.
* `inventory/`: Listen der Hardware- und Software-Inventarisierung.
